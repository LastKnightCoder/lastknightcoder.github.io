React 不支持在行内样式设置 `!important`

```jsx
// 无效
<p style={{fontSize: '20px !important'}}>Hello World</p>
```

<!--truncate-->

通过 `ref` 来设置

```jsx
<p ref={(node) => {
  if (node) {
    node.setProperty('font-size', '20px', 'important')
  }
}}>Hello World</p>
```
