部署YOLOv5训练

2022年5月17日 · 阅读需 4 分钟
熊滔
前端工程师 @ 高德

下载源码包，解压

wget https://github.com/ultralytics/yolov5/archive/refs/tags/v6.1.tar.gz
tar -zxvf yolov5-6.1.tar.gz

进入解压目录，安装依赖

cd yolov5-6.1
pip install -r requirements.txt

下载预训练模型 yolov5s6.pt，在项目根目录新建 weights 文件夹，将下载的预训练模型放入。
